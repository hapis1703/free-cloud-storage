# 🚀 Telegram Cloud Storage CLI

Simple CLI tool untuk upload & download file ke Telegram menggunakan:

- 🟢 Telegraf (Telegram Bot API)
- 🟢 better-sqlite3 (Database cepat & ringan)
- 🟢 Progress bar + Speed + ETA
- 🟢 Auto split file (18MB per chunk)

---

## 📦 Fitur

✅ Upload file besar (auto split 18MB per part)  
✅ Progress bar real-time  
✅ Speed (MB/s)  
✅ ETA  
✅ Simpan metadata (nama + ukuran file)  
✅ Download & merge otomatis  
✅ Hapus file dari database

---

## 📋 Requirements

- Node.js v18+
- Bot Telegram
- Channel Telegram (bot harus admin)
